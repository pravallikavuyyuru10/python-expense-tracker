# Python Expense Tracker

## About Project

This is a beginner Python project that demonstrates file handling by creating a simple expense tracker.

The program stores expenses in a text file and reads them whenever needed.

## Concepts Used

- Python File Handling
- Append Mode (`a`)
- Read Mode (`r`)
- Writing data to files
- Reading files line by line
- with open() statement

## How It Works

1. Opens expenses.txt file
2. Adds new expenses using append mode
3. Reads stored expenses
4. Displays expenses on the screen

## Sample Output

Food - 200  
Travel - 100  
Shopping - 500
